# 🚴 Bike-Drone — Drone-Guided Cycling Experience

## 📌 Project Overview
AeroRide is a drone-guided cycling event where participants follow a drone through nature without knowing the route in advance. A leader equipped with a screen and headset controls the drone and guides the group from point A to B.

## 🚀 How to Run
1. Clone the repo
2. Open `frontend/index.html` with Live Server
3. Fill in the registration form
4. Get redirected to the leaderboard

## 🎯 Key Features
- Participant registration (max 50 people)
- Strava leaderboard sorted by score
- AI-generated surprise routes
- BI dashboard for event analytics
- Drone-to-leader real-time communication
